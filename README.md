## Example Settings

The settings should be stored in the file `~/.tappedcv`

```clojure
{
  :threshold 0.7
  :height-buffer 175
  :dollarsign "/home/seabre/dollarsign.png"
  :android-screenshot-path "/sdcard/tappedcv-screenshot.png"
  :host-screenshot-path "/tmp/tappedcv-screenshot.png"
  :min-x 0
  :max-y 0
}
```
